    Predict Your Garment Worker Productivity

    Increase your garment factory's efficiency and output by predicting your worker's productivity. Our AI-powered platform uses advanced algorithms to analyze key factors such as worker experience, skill level, and performance history to generate accurate predictions.

    Reference Table for Input Features

    This table provides a reference for the different features or columns that you will be inputting values for. Use this as a guide to ensure that you are providing accurate and complete information for each feature.

    • Quarter

      A portion of the month. A month was divided into four quarters. You can input following values : 1,2,3 or 4

    • Department

      Department associated to each worker. You can choose between Sewing and Finishing.

    • Day of the week
      • Monday
      • Tuesday
      • Wednesday
      • Thursday
      • Saturday
      • Sunday

        • Friday is a holiday for workers.

    • Team Number

      Team identification number. Enter a number between 1 to 12.

    • Time Allocated

      Allocated time for a task(minutes). Enter a value between 2 to 54.

    • Unfinished Items

      Work in progress. Includes the number of unfinished items. Enter a value between 7 to 23122.

    • Over time

      Represents the amount of overtime by each team in minutes. Enter a value between 0 to 26000.

    • Incentive

      Represents the amount of financial incentive. Enter value between 0 to 3600.

    • Idle Time

      The amount of time(minutes) when the production was interrupted due to several reasons. Enter a value between 0 to 300.

    • Idle Men

      The number of workers who were idle due to production interruption. Enter a value between 0 to 45.

    • Style Change

      Number of changes in the style of a particular product. Enter a value between 0,1 or 2.

    • Number of Workers

      Number of workers in each team. Enter a value between 2 to 89.
